    LoL Statistics Tracker

    This project is a League of Legends statistics tracker that uses React, Next.js, Axios, and Express, and allows users to view their League of Legends player statistics and match history.

    To ensure a secure connection to the Riot API, I created a backend server with prevents users from accessing important secure information such as the Riot API key.

    This project is not yet available for use and is currently in development. I am focused on implementing other features such as multi-search user comparison and will work on design very soon.
